Help & Documentation

Selection Tool

Click and drag to select objects. Hold Shift to select multiple items. Drag the canvas background to pan.

Drawing Tools

Use the draw tool to create freeform paths. The membrane tool creates dashed membrane paths.

Version History

Commit your work regularly. Use the history slider to navigate between versions.

Components

Drag components from the sidebar onto the canvas. Double-click text to edit.
